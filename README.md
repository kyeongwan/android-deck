## 개요
매년 기술공유를 해주시는 발표자분들께 감사드립니다. 정말 훌륭한 발표가 많지만 흩어져있고, 추후에 찾아보려고 할 때 잘 찾을 수 없거나, 잊어버리는 경우가 있어서 한 곳에 정리합니다.

- 공유가 허락된 링크만 첨부합니다.
- repository에 파일 형식으로 업로드하지 않습니다.
